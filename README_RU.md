# Генератор преобразования структур Go в HashMap

Инструмент на Go, который автоматически генерирует код для преобразования структур Go в `map[string]any` (хэш-карту) на основе тегов структур. Поддерживает вложенные структуры и настраиваемое отображение полей.

## Возможности

- **Обнаружение тегов структур**: Использует пользовательский тег `structtomap` для указания, какие поля должны быть включены в карту.
- **Поддержка вложенных структур**: Рекурсивно преобразует вложенные поля структур во вложенные карты.
- **Генерация кода**: Создает готовые к использованию Go‑функции, выполняющие преобразование во время выполнения. **Одна функция на структуру** – даже если одна и та же структура встречается вложенной несколько раз, генерируется единая функция преобразования и используется повторно.
- **CLI и библиотека**: Может использоваться как инструмент командной строки или импортироваться как библиотека.
- **Нет зависимостей**: Чистый Go, без внешних зависимостей.

## Установка

```bash
go install github.com/andrey/gostructtohashmapgenerator/cmd/gostructtohashmap@latest
```

Или клонируйте репозиторий и соберите локально:

```bash
git clone https://github.com/andrey/gostructtohashmapgenerator.git
cd gostructtohashmapgenerator
go build ./cmd/gostructtohashmap
```

## Использование

### 1. Помечайте вашу структуру

Добавьте тег `structtomap` к полям, которые нужно включить. Значение тега может использоваться для указания пользовательского имени ключа; если оно пустое, используется имя поля.

```go
type Person struct {
    Name    string `structtomap:"name"`
    Age     int    `structtomap:"age"`
    Address struct {
        City    string `structtomap:"city"`
        Country string `structtomap:"country"`
    } `structtomap:"address"`
}
```

### 2. Сгенерируйте код преобразования

Запустите генератор для вашего исходного файла:

```bash
gostructtohashmap -input=person.go -output=person_map.go
```

Будет создан файл `person_map.go`, содержащий функцию `PersonToMap(p Person) map[string]any`. Если `Address` также является структурой, отдельная функция `AddressToMap` будет сгенерирована и вызвана внутри.

### 3. Используйте сгенерированную функцию

```go
p := Person{
    Name: "Алиса",
    Age:  30,
    Address: Address{City: "Москва", Country: "Россия"},
}
m := PersonToMap(p)
fmt.Printf("%#v", m)
// Вывод: map[string]any{"name":"Алиса", "age":30, "address":map[string]any{"city":"Москва", "country":"Россия"}}
```

## Расширенные настройки

- **Исключить непомеченные поля**: По умолчанию включаются только помеченные поля. Используйте `-all`, чтобы включить все поля.
- **Пользовательское имя тега**: Используйте `-tag=myTag` для поиска другого тега.
- **Глубина рекурсии**: Управляйте глубиной вложенности с помощью `-depth=N`.

## Использование как библиотеки

Вы также можете использовать генератор программно:

```go
import "github.com/andrey/gostructtohashmapgenerator/generator"

code, err := generator.Generate("person.go", generator.Options{
    Tag: "structtomap",
})
```
